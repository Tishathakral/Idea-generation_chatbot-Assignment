# Google Gemini Node.js Chatbot

Simple Google Gemini ChatBot for Nodejs

## Installation

```bash
npm install @google/generative-ai ora chalk prompt-sync
```

Replace the `API_KEY` in `index.js` with your own API key.

```bash
node index.js
```
